# 支持

Auto-Optimize Codex AGENTS.md 目前仅对 `main` 分支最新版本提供尽力而为的支持。

## 报告问题前

1. 拉取最新 `main` 分支并记录提交哈希。
2. 确认 Python 与 Git 可用，且 Codex 能读取仓库及你明确选择的会话目录。
3. 尽可能使用合成或脱敏数据复现。可用 [`submission/fixtures`](submission/fixtures/README.md) 中的生成器建立隔离环境。
4. 从日志和截图中移除密钥、私有源码、完整对话、用户名以及不必要的绝对路径。

## 获取支持

请提交 [GitHub Issue](https://github.com/ChanTinPing/auto-optimize-codex-agents-md/issues)，并包含：

- 仓库提交哈希；
- 操作系统、Python 版本和所用 Codex 界面；
- 使用模式（`Suggest` 或经明确授权的 `Auto`）；
- 最小复现步骤；
- 预期与实际行为；
- 已脱敏的错误输出或合成测试夹具。

不要在公开 Issue 中发布真实对话、凭据、API 密钥、私有代码或其他机密信息。

## 安全漏洞

疑似安全漏洞请勿提交公开 Issue。请通过 [GitHub 私密漏洞报告](https://github.com/ChanTinPing/auto-optimize-codex-agents-md/security/advisories/new) 私下报告；范围和要求见 [`SECURITY.md`](SECURITY.md)。

## 支持范围

本项目无法处理 OpenAI 账号访问、账单、身份验证、插件平台审核决定、Codex 服务可用性或 GitHub 账号管理问题。此类事项请联系对应服务商的官方支持渠道。

隐私详情见 [`PRIVACY.zh-CN.md`](PRIVACY.zh-CN.md)，使用条款见 [`TERMS.zh-CN.md`](TERMS.zh-CN.md)。
